An editor tool for procedurally generating pipes by just dragging the cursor from start to end; the pipe finds a way in a customizable way.

The tool was featured on 80.lv.

An editor tool for quickly switching the game's color palette without using any extra components, connecting directly to serialized properties and materials properties instead.
